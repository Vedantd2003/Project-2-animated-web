# Project 2 - Animated Web

A front-end animation practice build recreating a design-agency style landing page.

## What it is
A practice clone of a digital product design agency landing page ("LAZAREV."), built to work on scroll animations, custom fonts (Neue Haas Display), and SVG-based interactive elements. This is a learning/recreation exercise, not an original design.

## Demo
https://project-2-animated-web.vercel.app

## Tech Stack
- HTML5
- CSS3
- Vanilla JavaScript
- Locomotive Scroll
- Remix Icon
- Custom web fonts (Neue Haas Display)

## Features
- Animated hero section with morphing SVG icon
- Custom navigation with expandable menu sections
- Smooth-scroll page experience

## Local Setup
Static site — no build step required.

```bash
git clone https://github.com/Vedantd2003/Project-2-animated-web.git
cd Project-2-animated-web
# open index.html in a browser, or serve it:
npx serve .
```
